# Privacy Design

SavWise AI uses data minimisation. It collects only the demonstration data needed for financial coaching: profile values, consent records, transactions, goals, affordability checks, coach messages, and export logs.

The MVP must not collect real PINs, OTPs, bank credentials, national ID numbers, or confidential KYC documents.

Users can export JSON/CSV and delete/reset demo records. Future production versions would require stronger security controls, access management, encryption, provider approval, and regulatory review.
