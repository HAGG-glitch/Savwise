# Contributing

Use clear branches, meaningful commits, and testing before merge.

Recommended branches:

- `main`
- `joshua-integration`
- `bernard-dashboard-transactions`
- `moses-goals-affordability`

Commit examples:

- `Add transaction validation`
- `Calculate dashboard financial-health score`
- `Add Groq coach fallback response`
- `Document privacy design`

Do not commit API keys, personal financial data, database passwords, real mobile-money records, PINs, OTPs, or KYC documents.
